# SQLite

## Zweck
SQLite ist die Standarddatenbank der Anwendung und wird immer lokal verwendet.

## Eigenschaften
- Offline-First
- Plattformunabhängig
- Keine zusätzliche Installation erforderlich
- Grundlage für ChatDB, IndexDB, ProjectDB sowie Konfigurationsdaten.

## Verantwortlichkeiten
- Lokale Datenspeicherung
- Schneller Zugriff auf Konfigurationen
- Basis für spätere Synchronisation mit Remote-Systemen

SQLite ist verpflichtender Bestandteil der Architektur. Weitere Datenbanksysteme sind optionale Erweiterungen.
